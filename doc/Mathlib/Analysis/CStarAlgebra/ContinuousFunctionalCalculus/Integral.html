    Integrals and the continuous functional calculus #

    This file gives results about integrals of the form ∫ x, cfc (f x) a. Most notably, we show that the integral commutes with the continuous functional calculus under appropriate conditions.

    Main declarations #

    • cfc_integral: given a function f : X → 𝕜 → 𝕜, we have that cfc (fun r => ∫ x, f x r ∂μ) a = ∫ x, cfc (f x) a ∂μ under appropriate conditions
    • cfcₙ_integral: given a function f : X → 𝕜 → 𝕜, we have that cfcₙ (fun r => ∫ x, f x r ∂μ) a = ∫ x, cfcₙ (f x) a ∂μ under appropriate conditions
