    Idempotent completeness and functor categories #

    In this file we define an instance functor_category_isIdempotentComplete expressing that a functor category J тед C is idempotent complete when the target category C is.

    We also provide a fully faithful functor karoubiFunctorCategoryEmbedding : Karoubi (J тед C)) : J тед Karoubi C for all categories J and C.
