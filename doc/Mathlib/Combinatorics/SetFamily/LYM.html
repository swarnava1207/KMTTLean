    Lubell-Yamamoto-Meshalkin inequality and Sperner's theorem #

    This file proves the local LYM and LYM inequalities as well as Sperner's theorem.

    Main declarations #

    • Finset.local_lubell_yamamoto_meshalkin_inequality_div: Local Lubell-Yamamoto-Meshalkin inequality. The shadow of a set ùíú in a layer takes a greater proportion of its layer than ùíú does.
    • Finset.lubell_yamamoto_meshalkin_inequality_sum_card_div_choose: Lubell-Yamamoto-Meshalkin inequality. The sum of densities of ùíú in each layer is at most 1 for any antichain ùíú.
    • IsAntichain.sperner: Sperner's theorem. The size of any antichain in Finset Œ± is at most the size of the maximal layer of Finset Œ±. It is a corollary of lubell_yamamoto_meshalkin_inequality_sum_card_div_choose.
