    Positive Definite Matrices #

    This file defines positive (semi)definite matrices and connects the notion to positive definiteness of quadratic forms. Most results require ùïú = ‚Ñù or ‚ÑÇ.

    Main definitions #

    • Matrix.PosDef : a matrix M : Matrix n n ùïú is positive definite if it is hermitian and x·¥¥Mx is greater than zero for all nonzero x.
    • Matrix.PosSemidef : a matrix M : Matrix n n ùïú is positive semidefinite if it is hermitian and x·¥¥Mx is nonnegative for all x.

    Main results #

    • Matrix.posSemidef_iff_eq_transpose_mul_self : a matrix M : Matrix n n ùïú is positive semidefinite iff it has the form B·¥¥ * B for some B.
    • Matrix.PosSemidef.sqrt : the unique positive semidefinite square root of a positive semidefinite matrix. (See Matrix.PosSemidef.eq_sqrt_of_sq_eq for the proof of uniqueness.)
