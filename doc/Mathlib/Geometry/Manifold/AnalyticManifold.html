    Analytic manifolds (possibly with boundary or corners) #

    An analytic manifold is a manifold modelled on a normed vector space, or a subset like a half-space (to get manifolds with boundaries) for which changes of coordinates are analytic everywhere. The definition mirrors IsManifold, but using an analyticGroupoid in place of contDiffGroupoid. All analytic manifolds are smooth manifolds.
