    def Nat.lcm (m n : Nat) :
    Nat

    The least common multiple of m and n is the smallest natural number that's evenly divisible by both m and n. Returns 0 if either m or n is 0.

    Examples:

    • Nat.lcm 9 6 = 18
    • Nat.lcm 9 3 = 9
    • Nat.lcm 0 3 = 0
    • Nat.lcm 3 0 = 0
    Equations
    • m.lcm n = m * n / m.gcd n
