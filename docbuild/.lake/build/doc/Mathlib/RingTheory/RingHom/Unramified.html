    def RingHom.FormallyUnramified {R : Type u_1} {S : Type u_2} [CommRing R] [CommRing S] (f : R →+* S) :
    Prop

    A ring homomorphism R →+* A is formally unramified if Ω[A⁄R] is trivial. See Algebra.FormallyUnramified.
