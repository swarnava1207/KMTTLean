    Extended norm #

    In this file we define a structure ENormedSpace ùïú V representing an extended norm (i.e., a norm that can take the value ‚àû) on a vector space V over a normed field ùïú. We do not use class for an ENormedSpace because the same space can have more than one extended norm. For example, the space of measurable functions f : Œ± ‚Üí ‚Ñù has a family of L_p extended norms.

    • a NormedSpace structure on this space.

    The last definition is an instance because the type involves e.

    Implementation notes #

    We do not define extended normed groups. They can be added to the chain once someone will need them.
