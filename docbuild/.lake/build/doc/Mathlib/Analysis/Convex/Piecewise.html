    Convex and concave piecewise functions #

    This file proves convex and concave theorems for piecewise functions.

    Main statements #

    • convexOn_univ_piecewise_Iic_of_antitoneOn_Iic_monotoneOn_Ici is the proof that the piecewise function (Set.Iic e).piecewise f g of a function f decreasing and convex on Set.Iic e and a function g increasing and convex on Set.Ici e, such that f e = g e, is convex on the universal set.

      See convexOn_univ_piecewise_Ici_of_monotoneOn_Ici_antitoneOn_Iic for the version with the boundary point included in the right-hand function.
