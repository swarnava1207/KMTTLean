    Turing machine tapes #

    This file defines the notion of a Turing machine tape, and the operations on it. A tape is a bidirectional infinite sequence of cells, each of which stores an element of a given alphabet Γ. All but finitely many of the cells are required to hold the blank symbol default : Γ.

    Main definitions #

    • ListBlank Γ is the type of one-directional tapes with alphabet Γ. Implemented as a quotient of List Γ by extension by blanks at the end.
    • Tape Γ is the type of Turing machine tapes with alphabet Γ. Implemented as two ListBlank Γ instances, one for each direction, as well as a head symbol.
